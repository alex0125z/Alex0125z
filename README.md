* 自我的定义是有层次的。

  首先是相同的目标，其次是相同的记忆，然后是相同的性格，其次是相同的立场，最后是相同的肉体。

  好比洋葱。

  值得注意的是，性格与记忆并不重复。

  因为记忆对性格虽然有一定的影响，但也被身体中激素与药物等干涉着。


* 自我的定义也是模糊的。

  越往外层，对自我的界定也就越无价值。

  目标、记忆等完全相同，只有肉体不同的人，很多人也认为是“同一个人”。
